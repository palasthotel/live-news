== Description ==

Custom post type that let's you write new particle that will appear in frontend without the need of a reload.

== Installation ==

1. Upload `live-news.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `live-news` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a Live-News post
1. Change to "Live-Ticker" and start emitting content particles
